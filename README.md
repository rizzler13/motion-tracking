# MOTION TRACKING
This is developed as a part of open science initiative for perfusion imaging (OSIPI)'s Quality Control toolbox, for gsoc'26.
Motion tracking module in the qc tool for asl-mri

# To use it :
1. clone the repo
2. pip3 install nibabel nilearn nilearn numpy scipy matplotlib
3. python3 motiontracking.py "path to your file"
4. and run
   
this gives flags in the cli with the graphs generated in the 'outputs' folder
